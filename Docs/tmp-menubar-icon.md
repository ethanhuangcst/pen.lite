# Menu Bar Icon Functionalities

## 1. Menu Bar Icon Display
- Icon appearance
- Icon clickability
- Tooltip display
- Tooltip fade out


## 2. Online Mode Behaviors
- Display icon.png
- Display tooltip
- Left click behavior
  - Open Pen window
- Right click behavior
  - Show dropdown menu
    - Settings option
    - Exit option


## 3 Offline- Mode Behaviors
- Display icon_offline.png
- Display error tooltip
- Left click behavior
  - Show Reload option
  - Exit option
- Right click behavior
  - same as Left click behavior

- Manual logout transition
  - Online-login to online-logout
